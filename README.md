## Introducing Today's Project!

In this project, I set up a free tier AWS account on AWS.

### Key tools and concepts

Key tools I used: An authenticator app.
Key concepts I learned:
Creating an AWS account with root user credentials.
Enabling billing alerts to avoid going past the free tier.
Verifying my identity via SMS authentication.
Adding multi-factor authentication (MFA) to secure my root account.

### Challenges and wins

This project took me approximately 45 minutes. Nothing was too challenging — everything was manageable.

---

## Creating My AWS Account

In this step, I created an AWS account by setting up the account, verifying my email address, and creating a strong root user password.

### Understanding the root user

I learned that the root user is the main account user and has full access to the AWS account.

---

## Setting Up Billing and Payment

In this step, I set up billing by choosing a support plan and adding payment details.

### Why AWS requires a payment method

I learned that AWS requires a payment method to verify that you are a real person and not a bot trying to get free AWS resources, and to bill you if you go beyond the free tier limits.

---

## Completing Account Activation

In this step, I confirmed my identity and selected a support plan.

### Identity verification and support plan

I verified my identity by receiving a verification code on my phone and entering it on the AWS website. The Basic support plan is free and includes access to AWS documentation, whitepapers, and forums.

---

## Enabling Free Tier Alerts

In this step, I enabled free tier usage alerts to be notified when I get close to the free tier limit.

![Image](http://learn.nextwork.org/sincere_maroon_witty_monstera_deliciosa/uploads/aws-account-setup_j6k1l7m3)

### Why usage alerts matter

I set up usage alerts to be notified before exceeding my free tier usage limit, so I can avoid surprise bills.

---

## Securing the Account with MFA

To enable MFA, I scanned a QR code using an authenticator app. The app generates a 6-digit code that refreshes every 30 seconds, and you enter that code on the AWS website to complete the verification.
